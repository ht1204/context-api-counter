## Context API Counter

A simple counter using Context API with localstorage.
Contains a single interface with two buttons and input element:
- Increment the input value in one unit.
- Decrement the input value in one unit.

The value is preserved through the context loading the value using localtorage for \
initializing the value.

### Run the Project
Once you pull this project in your local
- As precondition, must have node v18.
- Install dependencies: `npm i` or `npm install`
- Run the Project: `npm run dev`
- Try clicking the buttons for incrementing and decrementing the value.
- Explorer the `CounterProvider` to check how it works.

Feel free to using and create your own version.

Have fun and enjoy!
